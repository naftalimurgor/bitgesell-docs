# clearbanned

`clearbanned`

Clear all banned IPs.

## Result

null (json null)

## Example usage (with RPC call)

```sh
BGL-cli clearbanned
```

```sh
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "clearbanned", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:8334/
```
